# دليل طبيبات مصر | المنصة الطبية الشاملة للمرأة المصرية

دليل طبيبات مصر هو منصة إلكترونية شاملة وموثوقة تهدف إلى تسهيل وصول السيدات والفتيات إلى الطبيبات (الإناث) في جميع المحافظات المصرية وعبر مختلف التخصصات الطبية. 

## 🌟 رؤية المنصة
تأسس هذا الدليل انطلاقاً من الإيمان بحق المرأة في الحصول على رعاية صحية تتسم بالخصوصية والراحة والمهنية الكاملة. تساعد المنصة في كسر حواجز البحث وتوجيه المريضات مباشرة إلى أفضل الكفاءات الطبية النسائية المتاحة في مناطقهن.

## ✨ مميزات المشروع
- **شامل ومفتوح لكافة التخصصات**: يضم الدليل طبيبات متخصصات في الجراحة، الباطنة، النساء والتوليد، الجلدية، الرمد، الطب النفسي، المخ والأعصاب، القلب، وغيرها من التخصصات الطبية المختلفة.
- **سهولة البحث والتصفية**: إمكانية البحث المباشر باسم الطبيبة، أو التصفية الذكية حسب المحافظة والحي أو التخصص الطبي لتسهيل الوصول لأقرب خيار مناسب.
- **تواصل مباشر عبر واتساب**: يتيح الدليل وسيلة تواصل سريعة وموحدة ومباشرة عبر تطبيق الواتساب لتسهيل الحجوزات والاستفسارات.
- **نظام موثوق للتحقق**: يتم مراجعة كافة الطلبات المقدمة من الطبيبات والتحقق من التراخيص المهنية والصور قبل اعتماد النشر لضمان حماية المريضات وموثوقية المعلومات.
- **لوحة تحكم إدارية متكاملة**: تتيح للمشرفين مراجعة الطلبات المعلقة، وإضافة وتحديث بيانات الطبيبات والمحافظات والمدن بكل مرونة وسهولة.
